# Agent Instructions

This repository is a protocol kit for responsible open-source contribution with
AI coding agents.

## Operating Rules

- Read `README.md` before editing primitives or examples.
- Treat files in `primitives/` as reusable public protocols. Keep them concise,
  portable, and tool-agnostic.
- Do not claim a contribution is merged, accepted, or maintainer-approved unless
  there is a public link proving it.
- Prefer source-backed examples over generic agent advice.
- Keep examples sanitized. Do not include private tokens, private issue context,
  company-confidential material, or personal data.
- When adding a new primitive, include:
  - when to use it
  - required inputs
  - output artifact
  - stop conditions
  - checklist

## Style

- Markdown-first.
- Short headings.
- Checklists over long prose.
- Explicit stop conditions.
- No tool-specific lock-in unless a section is clearly marked as an adapter.
